# Cypton：OKX 实盘行情监控与策略预警工具

原本的目的是想要自己写一个AI交易员，帮助进行AI分析和下单，类似量化引擎，但是比较困难，所以就写成了一个专门拉取实时数据的小工具，通过将拉取的信息进行指标的计算，然后保存数据信息打包为json，交给AI来进行分析。然后后续的话再进行一个和AI agent的对接。

Cypton 是一个针对 OKX 交易所开发的轻量级行情辅助工具。它的核心逻辑是：**实时抓取 K 线数据 -> 自动计算技术指标 -> 毫秒级策略扫描 -> 强弹窗/报警音提醒**。

它不是一个代你下单的黑盒脚本，而是一个帮你时刻盯着盘面并在关键点位“吼”你一声的看盘助手。

## 🛠️ 它能帮你做什么？

1.  **数据自动回补与多周期同步**
    * **历史回补**：启动程序时，系统会自动通过 REST API 补齐最近 800 根 K 线（LIMIT 可调），确保 MACD、布林带等长周期指标在第一秒就是准确的。
    * **多周期并行**：同时监听 1m、5m、15m、1h 等多个周期，每个周期独立维护缓存与指标计算。
    * **指标预计算**：内置 `pandas_ta`，实时生成 MACD、RSI、KDJ、布林带和成交量均线。

2.  **“防错过”多维预警机制**
    * **三种价格触发**：支持 `above`（破位）、`below`（跌穿）和 `reach`（触碰）。其中 `reach` 逻辑自带 0.05% 的容差判定，专门对付那些“差一点就挂到单”的尴尬行情。
    * **核弹级提醒**：触发时会调用 Tkinter 强制弹出置顶窗口，并伴随三声高分贝蜂鸣警报。在 Windows 和 macOS 下均有声音适配。
    * **异常保护存盘**：一旦触发预警，系统会立刻执行“强制落盘”，将当前的内存快照写入 JSON，防止因系统断电等极端情况丢失关键行情点位。

3.  **插件式策略扫描**
    * **背离识别**：扫描过去 60 根 K 线，寻找价格新高但 MACD 没新高的“顶背离”或价格新低但 MACD 没新低的“底背离”。
    * **轨道突破**：监控价格收盘穿透布林带上/下轨的瞬间信号。
    * **指标交叉**：识别标准的 MACD 金叉与死叉。

4.  **轻量化可视化**
    * 通过 Streamlit 提供 Web 看板。后端数据引擎运行的同时，前端会自动读取 `market_factory.json` 渲染 K 线图和信号点。

## 🏗️ 项目架构

* **`data_factory.py` (核心引擎)**：负责 WebSocket 长连接、多线程数据同步锁、指标计算及 `AlertManager` 预警逻辑。
* **`start_system.py` (一键启动)**：利用 `subprocess` 同时拉起后端数据源和前端 Streamlit 界面，并支持 Ctrl+C 优雅地同时关闭所有进程。
* **`config.py` (配置中心)**：管理交易所交易对（默认 ETH-USDT-SWAP）、代理地址、写入间隔等静态参数。
* **`strategies/` (策略库)**：存放独立的 Python 逻辑文件，方便用户自行扩展新的形态识别规则。

## 🚀 快速上手

1.  **环境安装**：
    确保已安装 `ccxt`、`pandas_ta`、`websocket-client` 和 `streamlit`。
2.  **配置代理**：
    打开 `config.py`，将 `PROXY_PORT` 修改为你本地代理（如 Clash）的端口。
3.  **运行程序**：
    ```bash
    python start_system.py
    ```
4.  **动态预警**：
    程序运行期间，你可以随时修改 `alerts.json` 里的价格和备注，系统会自动热加载配置，无需重启。
